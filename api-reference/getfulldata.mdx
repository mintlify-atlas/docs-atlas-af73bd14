## Overview

The `GetFullData` class fetches and parses comprehensive IP information from ipinfo.io, including geolocation, privacy detection, ASN details, abuse contacts, hosted domains, and more.

## Constructor

```python
GetFullData(ip, jwt_token)
```

Initializes and automatically fetches data for the given IP address.

<ParamField path="ip" type="str" required>
  The IP address to lookup (e.g., "8.8.8.8")
</ParamField>

<ParamField path="jwt_token" type="str" required>
  JWT token for authentication. Obtain this using the [GetJWTByLogin](/api-reference/getjwtbylogin) class.
</ParamField>

### Example

```python
from ipinfaux import GetFullData

data = GetFullData("8.8.8.8", "your-jwt-token")

if data.error:
    print(f"Error: {data.error}")
else:
    print(data.summary)
    print(data.geolocation)
```

## Methods

### as_dict()

Returns the complete data as a Python dictionary.

**Returns:** `dict` - All IP data

```python
data = GetFullData("8.8.8.8", jwt)
data_dict = data.as_dict()
print(data_dict["ip"])
print(data_dict["summary"]["company"])
```

### as_json(indent=2)

Returns the data as a formatted JSON string.

<ParamField path="indent" type="int" default="2">
  Number of spaces for JSON indentation
</ParamField>

**Returns:** `str` - JSON-formatted string

```python
data = GetFullData("8.8.8.8", jwt)
json_string = data.as_json(indent=4)
print(json_string)
```

### save(filename=None)

Saves the data to a JSON file.

<ParamField path="filename" type="str" default="None">
  Output filename. If not provided, defaults to `ipinfo_{ip}.json`
</ParamField>

**Returns:** `str | None` - Filename if successful, `None` if failed

```python
data = GetFullData("8.8.8.8", jwt)

# Save with auto-generated filename
data.save()  # Creates ipinfo_8.8.8.8.json

# Save with custom filename
data.save("custom.json")
```

### as_csv(include_headers=True, delimiter=',')

Returns the data as a CSV string with flattened keys.

<ParamField path="include_headers" type="bool" default="True">
  Whether to include column headers in the CSV output
</ParamField>

<ParamField path="delimiter" type="str" default="','">
  Character to use as field delimiter (e.g., ',', ';', '\t')
</ParamField>

**Returns:** `str` - CSV-formatted string

```python
data = GetFullData("8.8.8.8", jwt)
csv_string = data.as_csv()
print(csv_string)

# Without headers
csv_string = data.as_csv(include_headers=False)

# Tab-delimited
csv_string = data.as_csv(delimiter='\t')
```

<Note>
  Nested dictionaries are flattened using dot notation (e.g., `summary.asn`, `geolocation.city`). Lists are converted to JSON strings.
</Note>

### save_csv(filename=None, include_headers=True, delimiter=',')

Saves the data to a CSV file.

<ParamField path="filename" type="str" default="None">
  Output filename. If not provided, defaults to `ipinfo_{ip}.csv`
</ParamField>

<ParamField path="include_headers" type="bool" default="True">
  Whether to include column headers in the CSV file
</ParamField>

<ParamField path="delimiter" type="str" default="','">
  Character to use as field delimiter
</ParamField>

**Returns:** `str | None` - Filename if successful, `None` if failed

```python
data = GetFullData("8.8.8.8", jwt)

# Save with auto-generated filename
data.save_csv()  # Creates ipinfo_8.8.8.8.csv

# Save with custom filename and delimiter
data.save_csv("data.csv", delimiter=";")
```

## Complete Example

```python
from ipinfaux import GetFullData, GetJWTByLogin

# Get JWT token
jwt = GetJWTByLogin.login("your@email.com", "password")

if jwt:
    # Lookup multiple IPs
    ips = ["8.8.8.8", "1.1.1.1", "9.9.9.9"]
    
    for ip in ips:
        data = GetFullData(ip, jwt)
        
        if data.error:
            print(f"{ip}: Error - {data.error}")
        else:
            print(f"{ip}: {data.geolocation.get('city')}, {data.summary.get('company')}")
            
            # Save individual files
            data.save()
            data.save_csv()
            
            # Access specific data
            if data.privacy["vpn"]:
                print(f"  VPN detected: {data.privacy['details']}")
            
            print(f"  Hosted domains: {data.domains['count']}")
```
