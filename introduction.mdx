## What is ipinfaux?

ipinfaux is an unofficial Python library for scraping IP address information from ipinfo.io. It provides comprehensive IP geolocation, ASN details, privacy detection, and more through an authenticated scraping approach using JWT tokens.

Unlike API-based solutions, ipinfaux uses web scraping with BeautifulSoup to extract detailed IP information, making it ideal for projects that need rich IP data without API rate limits.

## What Data Can You Extract?

ipinfaux scrapes the following information for any IP address:

- **Summary** - ASN, hostname, company, IP range, RPKI validation
- **Geolocation** - City, state, country, postal code, timezone, coordinates
- **Privacy Detection** - VPN, proxy, Tor, hosting, relay detection
- **ASN Details** - Autonomous system number, name, domain, type
- **Abuse Contact** - Address, phone, email, network information
- **Hosted Domains** - List of domains hosted on the IP
- **Tags** - IP characteristics (CDN, hosting, anycast, etc.)
- **Alternative Data** - Cross-provider location validation

## System Requirements

<Note>
  **Python 3.7+** is required to use ipinfaux.
</Note>

The library depends on:
- `requests` >= 2.28.0 - HTTP client for web scraping
- `beautifulsoup4` >= 4.12.0 - HTML parsing engine

## Quick Example

Here's a minimal example to get you started:

```python
from ipinfaux import GetJWTByLogin, GetFullData

# Authenticate and get JWT token
jwt = GetJWTByLogin.login("your@email.com", "password")

# Lookup an IP address
data = GetFullData("8.8.8.8", jwt)

if not data.error:
    print(f"Location: {data.geolocation.get('city')}, {data.geolocation.get('country')}")
    print(f"Company: {data.summary.get('company')}")
    print(f"Hosting: {data.privacy.get('hosting')}")
```
