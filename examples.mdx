This page provides practical examples of how to use ipinfaux to lookup IP addresses and process the data.

## Basic Usage

<Tabs>
  <Tab title="Get Data as JSON">
    Retrieve IP information and work with it as JSON:

    ```python
    from ipinfaux import GetFullData

    # Lookup an IP address
    data = GetFullData("1.1.1.1", jwt)

    # Print JSON output
    print(data.as_json())

    # Save to default file: ipinfo_1.1.1.1.json
    data.save()

    # Save to custom filename
    data.save("custom.json")
    ```

    <Note>
      The `as_json()` method returns a formatted JSON string with 2-space indentation by default.
    </Note>
  </Tab>

  <Tab title="Get Data as CSV">
    Export IP information to CSV format:

    ```python
    from ipinfaux import GetFullData

    # Lookup an IP address
    data = GetFullData("8.8.8.8", jwt)

    # Print CSV output
    print(data.as_csv())

    # Save to default file: ipinfo_8.8.8.8.csv
    data.save_csv()

    # Save to custom filename with custom delimiter
    data.save_csv("data.csv", delimiter=",")
    ```

    <Note>
      CSV output flattens nested dictionaries using dot notation (e.g., `geolocation.city`).
    </Note>
  </Tab>
</Tabs>

## Processing Multiple IPs

Batch process multiple IP addresses and aggregate the results:

```python
from ipinfaux import GetFullData

# Define IP addresses to lookup
ips = ["8.8.8.8", "1.1.1.1", "9.9.9.9"]
results = []

# Process each IP
for ip in ips:
    data = GetFullData(ip, jwt)
    if not data.error:
        results.append(data)
        print(f"{ip}: {data.geolocation.get('city')}")

# Save each result individually
for result in results:
    result.save_csv(f"ipinfo_{result.ip}.csv")
```

**Example Output:**
```
8.8.8.8: Mountain View
1.1.1.1: Los Angeles
9.9.9.9: Berkeley
```

## Error Handling

Always check for errors when looking up IP addresses:

```python
from ipinfaux import GetFullData

data = GetFullData("8.8.8.8", jwt)

# Check if lookup failed
if data.error:
    print(f"Failed: {data.error}")
else:
    # Process data safely
    print(data.summary)
    print(data.geolocation)
    print(data.privacy)
```

<Note>
  Common errors include network timeouts, invalid JWT tokens, or parsing failures. Always check `data.error` before accessing other fields.
</Note>

## Authentication with Proxy

If you need to use a proxy for authentication:

```python
from ipinfaux import GetJWTByLogin

# Login with proxy
jwt = GetJWTByLogin.login(
    "your@email.com",
    "yourpassword",
    "http://proxy:8080"
)

if jwt:
    print("Login successful!")
    # Use jwt for lookups
else:
    print("Login failed")
```

## Complete Workflow Example

Here's a complete example that combines authentication and data processing:

```python
from ipinfaux import GetJWTByLogin, GetFullData
import json

# Step 1: Authenticate
jwt = GetJWTByLogin.login("your@email.com", "yourpassword")

if not jwt:
    print("Authentication failed")
    exit(1)

# Step 2: Lookup IP addresses
ips_to_check = ["8.8.8.8", "1.1.1.1", "9.9.9.9"]
results = []

for ip in ips_to_check:
    print(f"Looking up {ip}...")
    data = GetFullData(ip, jwt)
    
    if data.error:
        print(f"  Error: {data.error}")
        continue
    
    # Extract relevant information
    info = {
        "ip": ip,
        "city": data.geolocation.get("city"),
        "country": data.geolocation.get("country"),
        "company": data.summary.get("company"),
        "is_vpn": data.privacy["vpn"],
        "is_hosting": data.privacy["hosting"]
    }
    results.append(info)
    print(f"  Found: {info['city']}, {info['country']}")

# Step 3: Save results
with open("ip_report.json", "w") as f:
    json.dump(results, f, indent=2)

print(f"\nProcessed {len(results)} IPs successfully")
```

## Working with Tags

Check what services or characteristics are associated with an IP:

```python
from ipinfaux import GetFullData

data = GetFullData("8.8.8.8", jwt)

if not data.error:
    tags = data.tags
    
    # Find active tags
    active_tags = [tag for tag, active in tags.items() if active]
    
    print(f"Active tags: {', '.join(active_tags)}")
    
    # Check specific tags
    if tags['cdn']:
        print("This IP is a CDN")
    if tags['hosting']:
        print("This IP is a hosting service")
    if tags['anycast']:
        print("This IP uses anycast")
```

**Example Output:**
```
Active tags: anycast, cdn, hosting, webserver
This IP is a CDN
This IP is a hosting service
This IP uses anycast
```
