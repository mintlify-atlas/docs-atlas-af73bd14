## Get Started in 4 Steps

This guide will walk you through authenticating with ipinfo.io and performing your first IP lookup.

<Steps>
  <Step title="Install ipinfaux">
    If you haven't already, install ipinfaux using pip:

    ```bash
    pip install ipinfaux
    ```

    See the [Installation Guide](/installation) for more details.
  </Step>

  <Step title="Get JWT Token (Authentication)">
    Before looking up IP addresses, you need to authenticate with ipinfo.io to obtain a JWT token.

    <CodeGroup>
    ```python Basic Login
    from ipinfaux import GetJWTByLogin

    # Login with your ipinfo.io credentials
    jwt = GetJWTByLogin.login("your@email.com", "yourpassword")

    if jwt:
        print("Login successful!")
    else:
        print("Login failed")
    ```

    ```python With Proxy
    from ipinfaux import GetJWTByLogin

    # Login through a proxy
    jwt = GetJWTByLogin.login(
        "your@email.com",
        "yourpassword",
        "http://proxy:8080"
    )
    ```
    </CodeGroup>

    <Warning>
      You need valid ipinfo.io account credentials. Sign up at [ipinfo.io](https://ipinfo.io) if you don't have an account.
    </Warning>
  </Step>

  <Step title="Lookup an IP Address">
    Once you have a JWT token, use it to look up any IP address:

    ```python
    from ipinfaux import GetFullData

    # Lookup Google's DNS server
    data = GetFullData("8.8.8.8", jwt)

    # Check if the lookup was successful
    if data.error:
        print(f"Error: {data.error}")
    else:
        print("Lookup successful!")
    ```

    The `GetFullData` class automatically fetches and parses all available information about the IP address.
  </Step>

  <Step title="Access the Data">
    The data is organized into structured attributes that you can easily access:

    ```python
    # Access different data categories
    print(f"IP: {data.ip}")
    print(f"\nSummary: {data.summary}")
    print(f"\nGeolocation: {data.geolocation}")
    print(f"\nPrivacy Detection: {data.privacy}")
    print(f"\nASN Details: {data.asn}")
    print(f"\nAbuse Contact: {data.abuse}")
    ```

    <Tip>
      Each attribute returns a dictionary with structured data. Use `.get()` to safely access nested values.
    </Tip>
  </Step>
</Steps>

## Complete Working Example

Here's a complete script that demonstrates the full workflow:

```python title="example.py" icon="python"
from ipinfaux import GetJWTByLogin, GetFullData

# Step 1: Authenticate
jwt = GetJWTByLogin.login("your@email.com", "yourpassword")

if not jwt:
    print("Failed to login")
    exit(1)

# Step 2: Lookup an IP
data = GetFullData("8.8.8.8", jwt)

# Step 3: Check for errors
if data.error:
    print(f"Lookup failed: {data.error}")
    exit(1)

# Step 4: Access and display data
print(f"IP Address: {data.ip}")
print(f"\nLocation:")
print(f"  City: {data.geolocation.get('city')}")
print(f"  State: {data.geolocation.get('state')}")
print(f"  Country: {data.geolocation.get('country')}")

print(f"\nCompany: {data.summary.get('company')}")
print(f"Hostname: {data.summary.get('hostname')}")
print(f"ASN: {data.summary.get('asn')}")

print(f"\nPrivacy:")
print(f"  Hosting: {data.privacy.get('hosting')}")
print(f"  VPN: {data.privacy.get('vpn')}")
print(f"  Proxy: {data.privacy.get('proxy')}")
```

## Export Data

ipinfaux provides built-in methods to export data in different formats.

<CodeGroup>
```python JSON Export
from ipinfaux import GetFullData

data = GetFullData("1.1.1.1", jwt)

# Print as JSON
print(data.as_json())

# Save to default file (ipinfo_1.1.1.1.json)
data.save()

# Save to custom file
data.save("cloudflare.json")
```

```python CSV Export
from ipinfaux import GetFullData

data = GetFullData("8.8.8.8", jwt)

# Print as CSV
print(data.as_csv())

# Save to default file (ipinfo_8.8.8.8.csv)
data.save_csv()

# Save with custom delimiter
data.save_csv("data.csv", delimiter=",")
```
</CodeGroup>

## Process Multiple IPs

Lookup multiple IP addresses and batch export to CSV:

```python title="batch_lookup.py" icon="python" lines="1-5,10-12"
from ipinfaux import GetJWTByLogin, GetFullData

# Authenticate
jwt = GetJWTByLogin.login("your@email.com", "password")

# List of IPs to lookup
ips = ["8.8.8.8", "1.1.1.1", "9.9.9.9"]
results = []

# Process each IP
for ip in ips:
    data = GetFullData(ip, jwt)
    if not data.error:
        results.append(data)
        print(f"{ip}: {data.geolocation.get('city')}")

# Save each result individually
for result in results:
    result.save_csv(f"ipinfo_{result.ip}.csv")
print(f"\nExported {len(results)} results to CSV files")
```

## Error Handling

Always check for errors before accessing data:

```python
data = GetFullData("8.8.8.8", jwt)

if data.error:
    print(f"Error occurred: {data.error}")
    # Handle the error appropriately
else:
    # Safe to access data
    print(data.summary)
```

Common error scenarios:
- **Invalid JWT token**: Authentication failed or token expired
- **Network errors**: Connection timeout or DNS resolution failure
- **Parsing errors**: HTML structure changed (rare)
